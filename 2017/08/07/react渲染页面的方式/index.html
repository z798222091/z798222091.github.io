react渲染页面的方式

react渲染页面的方式

  1. 上边的代码中,创建了一个叫做Header的类,而我们所创建的每一个类都是一个组件,在这个类中有一个render方法,它返回的是一个dom结构.而在最外面要用一个jsx元素把里面的内容包裹起来;
  2. 在上边我们在react-dom 中引用了ReactDom这个属性,而这个属性中包含了一个render方法,这个方法接收两个参数,第一个参数是我们创建好的组件,里面包含了我们定义好的dom结构,第二个参数代表我们要插入的dom元素.通过这种方法,我们可以把我们需要的dom结构插入到特定的dom元素中;

表达式插入

  • 我们也可以将js的表达式插入到我们所写的组件中,表达式要用{}包起来,而表达式的的结果也会渲染到dom结构中去;

  • 而作为表达式,里面是可以放任意类型的js代码,它都会把结果渲染到我们的页面上,所以说这个表达式是很灵活的;比如:根据不同的判断结果返回不同的值;
