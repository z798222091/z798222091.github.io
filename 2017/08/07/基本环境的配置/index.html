react开发环境的配置

react

关于对react的理解: react就是帮助我们把界面分成了好多个小块,每一个小块都是一个组件,组件之间可以嵌套,可以组合,就形成了我们的页面.
其实react.js只是一个库,而不是一个框架,它只是帮我们解决UI层面的问题,而我们要操作数据的变化等的操作,就要借助redux,react-router等来协助了.

基本环境的配置:

1.首先,使用官方推荐的工具 reacte-react-app 直接生成我们所需要的工程目录,
这个工程目录会帮我们做好各种配置和依赖;

create-react-app xxxxx(项目名称)

yarn start || npm start
```
